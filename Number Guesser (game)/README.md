This is a program that:
a.) Prompts the user for a level, 𝑛. If the user does not input a positive integer, the program prompts again.
b.) Randomly generates an integer between 1 and 𝑛, inclusive, using the random module.
c.) Prompts the user to guess that integer. If the guess is not a positive integer, the program prompts the user again.
  (c.1) If the guess is smaller than that integer, program outputs Too small! and prompts the user again.
  (c.2) If the guess is larger than that integer, the program outputs Too large! and prompts the user again.
  (c.3) If the guess is the same as that integer, the program outputs Just right! and exit.
